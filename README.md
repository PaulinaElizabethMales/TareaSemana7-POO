# Proyecto Género de Animes - Semana 7

Este programa demuestra el uso de **constructores (__init__)** y **destructores (__del__)** en Python,
utilizando una arquitectura de **Modelos/Servicios**.

## Estructura
- `modelos/` → contiene la clase `GeneroAnime`.
- `servicios/` → contiene la clase `GestorGeneros`.
- `main.py` → punto de entrada del programa.

## Uso de Constructores y Destructores en Python
- Al crear un género, se ejecuta automáticamente el constructor (__init__).
- Al eliminar un género, se ejecuta el destructor (__del__), mostrando un mensaje de limpieza.
